##  Computer Lab Finder App

An Android mobile application that helps university students and staff easily find, explore, and book computer laboratories within a university campus.

---

###  Problem Addressed

Students waste time searching for computer labs due to:
- Labs being full or closed
- No centralized lab information
- Difficulty identifying suitable labs

This app provides **quick, reliable, and offline access** to lab details.

---

###  Key Features

- User registration & login
- View available computer labs
- Lab details (capacity, equipment, operating hours)
- Lab booking & booking management
- Explore labs by location and software
- Google Maps directions
- Device orientation sensor display
- Profile management & user reviews
- Offline access using SQLite

---

###  Tech Stack

- **Language:** Java  
- **IDE:** Android Studio  
- **UI:** XML  
- **Database:** SQLite  
- **Build Tool:** Gradle
